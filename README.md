# 🛒 Retail Store Sales Analysis

## 📌 Overview

This project analyzes Retail Store Sales data using Python to find useful business insights and trends.

---

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📊 Analysis Performed

* Data Cleaning
* Sales Analysis
* Profit Analysis
* Payment Mode Analysis
* Customer Analysis
* Monthly Sales Trend

---

## 📈 Key Insights

* Clothing category has highest sales.
* COD is the most used payment mode.
* Maharashtra generated high sales.
* Festival months showed better sales growth.
